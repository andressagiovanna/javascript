Relacionais

Pra toda expressão que tenha o operador relacional ligado a ela, o resultado dessa expressão vai ser sempre um valor BOLEANO -> Verdadeiro / Falso

- 5 > 2 -> true
- 7 < 4 -> false
- 8 <= 8 -> true ( <= -> MAIOR OU IGUAL )
- 9 <= 7 -> false ( <= -> MENOR OU IGUAL )
- 5 == 5 -> true ( == -> IGUALDADE )
- 4 != 4 -> false ( != -> NÃO IGUAL / DIFERENTE )

- preço >= 200.50 -> o preço é maior ou igual a 200.50?
- idade < 18 -> a idade é menor do que 18?
- curso == 'JavaScript' -> o curso é JavaScript?
- n1 != n2 -> o primeiro número é diferente do segundo?

Identidade

Operador de Igualdade Restrita

- 5 == 5 -> true
- 5 == '5' -> true
- 5 === '5' -> false ( === -> IDENTICOS, mesmo valor e mesmo tipo)
- 5 === 5 -> true

- var x = 5
- var y = '5'
- x !== y -> true ( !== -> DESIGUAL RESTRITO, tipos diferentes, valor interno igual)

Lógicos

- ! -> NEGAÇÃO
*** Operador Unário: Só tem 1 operando;
*** ! -> TRUE (uma coisa que não é VERDADEIRA é FALSA) / FALSE (uma coisa que não é FALSA é VERDADEIRA)

- && -> CONJUNÇÃO ( Êlógico )
*** Operador Binário *** true && true -> TRUE
*** true && false -> FALSE
*** false && true -> FALSE
*** false && false -> FALSE

- || -> DISJUNÇÃO ( OUlógico )
*** Operador Binário *** true || true -> TRUE
*** true || false -> TRUE
*** false || true -> TRUE
*** false || false -> FALSE

1º OPERADOR ARITMÉTICO
2º OPERADOR RELACIONAIS (quem aparecer primeiro vai ser feito primeiro / da esquerda para a direita)
3º OPERADOR LÓGICOS

1º NÃO (!)
2º Ê (&&)
3º OU (||)

Exemplo

- idade >= 15 && idade <= 17 -> a idade está entre 15 e 17 ?
- estado == 'RJ' || estado == 'SP' -> o estado é RJ ou SP?
- salário > 1500 && sexo != 'M'-> o salário é acima de 1500 e não é homem?

Ternário

Junta três operandos

- teste ? true : false
- média >= 7.0 ? "Aprovado" : "Reprovado"
