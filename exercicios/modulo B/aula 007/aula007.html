Aritméticos

- 5 + 2 -> 7
- 5 - 2 -> 3
- 5 * 2 -> 10
- 5 / 2 -> 2.5 (Para o JS é uma divisão REAL, que leva a vírgula)
- 5 % 2 -> 1 (% -> Resto da divisão inteira)
- 5 ** 2 -> 25 (** -> Potência)

Precedência

- () -> 1º PARÊNTESES
- ** -> 2º POTÊNCIA
- * / % -> 3º MULTIPLICAÇÃO, DIVISÃO e o RESTO DA DIVISÃO (valendo o que vem da esquerda pra direita)
- + - -> 4º ADIÇÃO, SUBTRAÇÃO

Atribuição Simples

var a = 5 + 3 -> 8
var b = a % 3 -> 3
var c = 5 * b ** 2 -> 45
var d = 10 - a / 2 -> 6
var e = 6 * 2 / d -> 2
var f = b % e + 4 / e -> 3

Auto-atribuição

var n = 3

n = n + 4 -> n = 7
n = n - 5 -> n = 2
n = n * 4 -> n = 8
n = n / 2 -> n = 4
n = n ** 2 -> n = 16
n = n % 5 -> n = 1

Auto-referência (SIMPLIFICADO)

var n = 3

n = n + 4 -> n += 4
n = n - 5 -> n -= 5
n = n * 4 -> n *= 4
n = n / 2 -> n /= 2
n = n ** 2 -> n **= 2
n = n % 5 -> n %= 5

Incremento

var x = 5

x = x + 1 -> ++ x (Pré)
x = x - 1 -> -- x (Pré)

x = x + 1 -> x ++ (Pós)
x = x - 1 -> x -- (Pós)
