Em JS

// -> Uma única Linha
/* */ -> Mais de uma linha de código

Exempos

- vaga a1 = carro1 -> 1 Sinal de igual em JavaScript é o mesmo que recebe.
- vaga a1 = carro2
- vaga a1 = null -> Nulo, sem carro

*** No JavaScript moderno, além de utilizar a palavra var, também podemos usar a palavra let.

- var n1 = 5
- var n2 = 8.5
- var n3 = 15

- var s1 = "JavaScript"
- var s2 = 'Curso em Vídeo'
- var s3 = `Guanabara`

*** Nome de cada variável -> IDENTIFICADOR
- Podemos começar com letra, $ ou _.
- Não podemos começar com Números.
- É possível usar Letras ou Números.
- É possível usar Acentos e Símbolos.
- Não podem conter Espaços.
- Não podem ser palavras reservadas (Varvar).

Dicas

- Maiúsculas e Minúsculas fazem diferença.
- Tente escolher nomes coerentes para as variáveis.
- Evite se tornar um 'Programador Alfabeto' ou um 'Programador Contador'.

Tipos Primitivos Primordiais - DATA TYPES

- 5, 18, -12 -> Número Inteiro
- 0.5, -15.9, 3.14, 8.0 -> Números Reais (Com ponto Flutuante - flout)
TIPO NUMBER

- true, false -> Valor Boleano (BOOLEAN)

- "Google", 'JavaScript', `Maria` -> Cadeia de caracteres (STRING).
- As strings também podem ser um conjunto de números:
-> CPF (000.000.000-00)
-> Telefone ({00}0 0000-0000)
-> Identidade (SP 00.000.000)

TYPEOF:
- Number -> Infinity e NaN (Not a Number)
- String
- Boolean
- Null
- Undefined
- Object -> Array (Null)
- Function
